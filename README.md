# Python 100 Days Code
This project contains my progress through the 100 Days of Code Python Udemy course. The goal of the challenge is to improve my Python skills by coding a lot of variety of projects and strengthening my problem-solving skills.

Throughout this journey, I wil be covering:
- Python fundamentals
- Object Oriented Programming (OOP)
- Data structures and algorithms
- File handling
- APIs and web requests


## Projects included
Each folder represents a day of the challenge. These are some of the main projects that will be built.

- Quiz
- Snake and Pong Game
- Web scraper

## Technologies Used
- PyCharm
- Python libraries (turtle, pandas, flask)
